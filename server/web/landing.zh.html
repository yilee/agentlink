使用方式

三步，就这么简单。

01

启动 Agent

在项目目录下运行 agentlink-client start -D，终端会打印一个会话链接。

02

在任意设备打开链接

手机、平板、别人的电脑 — 只要有浏览器就行。所有通信自动端到端加密，中继服务器看不到任何内容。

03

开始干活

多个对话并行运行，随时切换项目。完整的会话历史随时可以恢复，工作不会丢。
