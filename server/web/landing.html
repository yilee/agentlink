AgentLink

Remote Control for
Claude Code

Start a task in your terminal, close the laptop and go for a walk — then pick it up from your phone, a tablet, or any browser. The session stays alive on your machine.

Get started in 30 seconds
1
npm install -g @agent-link/agent click to copy
2
agentlink-client start -D click to copy
Open the URL it prints. That's it.

How it works

💻

Start the agent on your dev machine

Run agentlink-client start -D in your project directory. It prints a URL like https://msclaude.ai/s/abc123.

🔒

The relay encrypts everything

Your messages and code travel through the relay server, but it can't read any of it. All traffic is end-to-end encrypted with XSalsa20-Poly1305. The server is a stateless, blind forwarder.

📱

Open the URL on any device

Your phone, a tablet, a colleague's laptop — any browser works. Kick off a refactoring task from your desk, then monitor progress from the couch.

🔄

Pick up where you left off

Full session history from Claude Code's own logs. Resume any past conversation, switch projects, change working directories — all from the browser.

Features

⚡

Zero config

Install, start, open the URL. No accounts, no registration, no configuration files.

🔒

Optional password protection

Add --password to lock your session. Brute-force protection built in.

📄

Full Claude Code

File editing, bash commands, search, multi-turn conversations — everything Claude Code can do, in your browser.

🌐

Works on any device

Responsive UI for phones, tablets, and desktops. Use Claude Code from anywhere with a browser.

🔄

Auto-reconnect

Laptop sleeps, Wi-Fi drops, network switches — the agent reconnects automatically when it's back online. No lost sessions.

🚧

Self-hostable

Run your own relay server with @agent-link/server. Full control over your infrastructure.
